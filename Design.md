# AI 履歷產生器 App 設計文件

## 一、專案名稱
AI 履歷產生器 App

## 二、專案目的
本專案希望透過簡單的網頁介面，讓使用者輸入姓名、科系、技能與專題經驗後，自動產生一份履歷草稿，協助學生快速整理個人資料與專題成果。

## 三、使用者需求
- 使用者可以輸入基本資料
- 使用者可以輸入專業技能
- 使用者可以輸入專題名稱
- 系統可以自動產生履歷文字
- 網頁介面需簡單、清楚、容易操作

## 四、主要功能
1. 姓名輸入
2. 科系輸入
3. 技能輸入
4. 專題經驗輸入
5. 履歷草稿自動產生

## 五、UI/UX 設計
本系統採用卡片式版面，分為標題區、輸入區、按鈕區與結果顯示區。使用者可以依照由上到下的流程完成操作，降低學習成本。

## 六、技術架構
- Frontend：Next.js
- Language：TypeScript
- UI：Tailwind CSS
- Version Control：GitHub
- Deployment：Railway

## 七、部署方式
專案完成後上傳至 GitHub Repository，再透過 Railway 連接 GitHub 專案，自動建置並產生公開網址。

## 八、未來改進
未來可串接真正的 AI API，例如 OpenAI API，讓履歷內容更自然，也可以加入 PDF 匯出、模板選擇與中英文履歷切換功能。
